===================================================
   Syndicate Management System - Instructions
===================================================

1. Install Node.js from https://nodejs.org/ (if not already installed).
2. Extract the contents of this ZIP file to a folder on your computer.
3. FIRST TIME ONLY: Double-click "setup_and_run.bat" to install libraries.
4. EVERY OTHER TIME: Double-click "run_app.bat" to start the program immediately.
5. The application will be available at:
   http://localhost:3000
   OR
   http://192.168.0.118:3000

To access from other devices on the same network:
- Open the browser on the other device and go to: http://192.168.0.118:3000

Enjoy using the Syndicate Management System!
